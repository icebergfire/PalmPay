# PalmPay

Биометрическая система бесконтактного доступа и оплаты по ладони.

PalmPay использует инфракрасное сканирование вен ладони для идентификации пользователя. Система заменяет пластиковые пропуска и банковские карты единым биометрическим идентификатором — рисунком вен, который невозможно подделать или передать другому человеку.

Приложение построено на технологиях компьютерного зрения (MediaPipe Hands, MobileNet v2) с трёхканальной биометрической фузией (геометрия, визуал, текстура) и встроенной пассивной проверкой живости (Liveness Detection).

Первое внедрение планируется на турникетах университетского кампуса с последующей интеграцией с банковскими платёжными терминалами.

## Структура

```
PalmPay/
├── mobile/
│   ├── palmpay.html    # PWA-приложение (UI + биометрия + камера)
│   ├── manifest.json   # Web App Manifest
│   └── sw.js           # Service Worker
└── docs/               # Документация
```

## Запуск

```bash
git clone https://github.com/icebergfire/PalmPay.git
cd PalmPay
python3 -m http.server 8080
```

Откройте `http://localhost:8080/mobile/palmpay.html` в браузере.
